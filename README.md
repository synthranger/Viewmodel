# Viewmodel
Viewmodel class for handling viewmodels in FPS frameworks

Viewmodel Instance Format
- Model (Model)
  - HumanoidRootPart (Part) (Model.PrimaryPart)
  - FakeCamera (Part)
  - Settings (ModuleScript) (Returns ViewmodelSettings)
  - Left Arm (Part)
  - Right Arm (Part)
  - AnimationController or Humanoid
